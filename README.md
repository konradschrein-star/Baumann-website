# DER BAUMANN - Webseiten Präsentation

## Übersicht

Diese Landing-Page zeigt drei verschiedene Webseiten-Designs für "DER BAUMANN" Terrassenbau & Handwerk.

## Struktur

```
DER BAUMANN/
├── index.html          # Haupt-Landing-Page mit Design-Auswahl
├── styles.css          # Styling für die Landing-Page
├── design1/            # Erste Webseite (bereits fertig)
│   ├── index.html
│   ├── assets/
│   └── pages/
├── design2/            # Zweite Webseite (Platzhalter)
│   └── index.html
└── design3/            # Dritte Webseite (Platzhalter)
    └── index.html
```

## Wie es funktioniert

1. **Landing-Page (index.html)**:
   - Zeigt drei große Browser-Preview-Windows
   - Kunde kann zwischen den Designs wählen
   - Klick auf "Design ansehen" öffnet die entsprechende Webseite

2. **Design 1**:
   - Vollständige Premium-Website bereits implementiert
   - Terrassenbau-fokussiertes Design

3. **Design 2 & 3**:
   - Aktuell Platzhalter
   - Zeigen "Folgt in Kürze" Meldung

## Neue Designs hinzufügen

### Für Design 2:
1. Ersetze den Inhalt von `design2/index.html` mit deiner neuen Webseite
2. Kopiere alle Assets (CSS, JS, Bilder) in den `design2/` Ordner
3. Die Landing-Page zeigt automatisch eine Vorschau an

### Für Design 3:
1. Ersetze den Inhalt von `design3/index.html` mit deiner neuen Webseite
2. Kopiere alle Assets (CSS, JS, Bilder) in den `design3/` Ordner
3. Die Landing-Page zeigt automatisch eine Vorschau an

## Features der Landing-Page

- ✅ Responsive Design (funktioniert auf allen Geräten)
- ✅ Browser-Mockup Preview-Windows
- ✅ Live-Vorschau im iframe
- ✅ Hover-Effekte und Animationen
- ✅ Professionelles Gradient-Design
- ✅ Automatische Deaktivierung für nicht-verfügbare Designs

## Öffnen der Landing-Page

Öffne einfach die `index.html` Datei im Browser, um die Design-Auswahl zu sehen.

## Tipps

- Jedes Design sollte in seinem eigenen Ordner bleiben
- Die Ordner design1, design2, design3 sollten eigenständige Webseiten enthalten
- Die Landing-Page (index.html im Hauptverzeichnis) zeigt alle Designs automatisch an
